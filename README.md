# GapLens AI

GapLens AI is an AI-powered web application that helps learners identify prerequisite concepts before studying advanced technical topics.

## Features

- AI-generated prerequisite learning roadmap
- Ordered learning path
- Common beginner mistakes
- Simple and user-friendly interface

## Tech Stack

- HTML5
- CSS3
- Bootstrap 5
- JavaScript (ES6)
- Google Gemini 3.1 Flash Lite API
- Vercel Serverless Functions
- Vercel

## Setup Instructions

1. Clone the repository.
2. Open the project in Visual Studio Code.
3. Create a Vercel Environment Variable named `GEMINI_API_KEY` and add your Gemini API key.
4. Deploy the project on Vercel.
5. Open the deployed application and start using GapLens AI.
